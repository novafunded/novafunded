# NovaFunded

NovaFunded is a demo prop trading challenge platform.

Traders can purchase a $5,000 demo challenge for $11 and prove their trading ability.

## Challenge Rules

- Starting Balance: $5,000
- Profit Target: 8% ($400)
- Maximum Daily Loss: 5% ($250)
- Maximum Drawdown: 10% ($500)
- Minimum Trading Days: 3
- Challenge Duration: 30 days

## How It Works

1. Purchase the $5k demo challenge for $11
2. Trade on a demo account
3. Follow the risk rules
4. Reach the 8% profit target
5. Pass the challenge

Traders who pass will receive priority access to NovaFunded real funded accounts when the platform launches.

## Tech Stack

- Next.js
- TailwindCSS
- Firebase
- Stripe
- Vercel
